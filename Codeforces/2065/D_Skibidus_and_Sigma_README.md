<h3><a href="https://codeforces.com/contest/2065/problem/D" target="_blank">Skibidus and Sigma</a></h3>

<div class="header"><div class="title">D. Skibidus and Sigma</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Let's denote the score of an array $$$b$$$ with $$$k$$$ elements as $$$\sum_{i=1}^{k}\left(\sum_{j=1}^ib_j\right)$$$. In other words, let $$$S_i$$$ denote the sum of the first $$$i$$$ elements of $$$b$$$. Then, the score can be denoted as $$$S_1+S_2+\ldots+S_k$$$.</p><p>Skibidus is given $$$n$$$ arrays $$$a_1,a_2,\ldots,a_n$$$, each of which contains $$$m$$$ elements. Being the sigma that he is, he would like to concatenate them in <span class="tex-font-style-bf">any order</span> to form a single array containing $$$n\cdot m$$$ elements. Please find the maximum possible score Skibidus can achieve with his concatenated array! </p><p>Formally, among all possible permutations$$$^{\text{∗}}$$$ $$$p$$$ of length $$$n$$$, output the maximum score of $$$a_{p_1} + a_{p_2} + \dots + a_{p_n}$$$, where $$$+$$$ represents concatenation$$$^{\text{†}}$$$.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A permutation of length $$$n$$$ contains all integers from $$$1$$$ to $$$n$$$ exactly once.</p><p>$$$^{\text{†}}$$$The concatenation of two arrays $$$c$$$ and $$$d$$$ with lengths $$$e$$$ and $$$f$$$ respectively (i.e. $$$c + d$$$) is $$$c_1, c_2, \ldots, c_e, d_1, d_2, \ldots d_f$$$.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains an integer $$$t$$$ ($$$1 \leq t \leq 10^4$$$) — the number of test cases.</p><p>The first line of each test case contains two integers $$$n$$$ and $$$m$$$ ($$$1 \leq n \cdot m \leq 2 \cdot 10^5$$$) — the number of arrays and the length of each array.</p><p>The $$$i$$$'th of the next $$$n$$$ lines contains $$$m$$$ integers $$$a_{i,1}, a_{i,2}, \ldots, a_{i,m}$$$ ($$$1 \leq a_{i,j} \leq 10^6$$$) — the elements of the $$$i$$$'th array.</p><p>It is guaranteed that the sum of $$$n \cdot m$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output the maximum score among all possible permutations $$$p$$$ on a new line.</p></div><div class="sample-tests"><div Then, $$$a_{p_1} + a_{p_2} = [4, 4, 6, 1]$$$. Its score is $$$4+(4+4)+(4+4+6)+(4+4+6+1)=41$$$. </li><li> $$$p = [2, 1]$$$. Then, $$$a_{p_1} + a_{p_2} = [6, 1, 4, 4]$$$. Its score is $$$6+(6+1)+(6+1+4)+(6+1+4+4)=39$$$. </li></ul><p>The maximum possible score is $$$41$$$.</p><p>In the second test case, one optimal arrangement of the final concatenated array is $$$[4,1,2,1,2,2,2,2,3,2,1,2]$$$. We can calculate that the score is $$$162$$$.</p></div>
