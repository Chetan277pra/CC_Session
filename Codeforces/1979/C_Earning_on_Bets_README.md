<div class="header"><div class="title">C. Earning on Bets</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You have been offered to play a game. In this game, there are $$$n$$$ possible outcomes, and for each of them, you must bet a certain <span class="tex-font-style-bf">integer</span> amount of coins. In the event that the $$$i$$$-th outcome turns out to be winning, you will receive back the amount of coins equal to your bet on that outcome, multiplied by $$$k_i$$$. Note that <span class="tex-font-style-bf">exactly one</span> of the $$$n$$$ outcomes will be winning.</p><p>Your task is to determine how to distribute the coins in such a way that you will come out ahead in the event of <span class="tex-font-style-bf">any</span> winning outcome. More formally, the total amount of coins you bet on all outcomes must be <span class="tex-font-style-bf">strictly less</span> than the number of coins received back for each possible winning outcome.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test consists of multiple test cases. The first line contains a single integer $$$t$$$ ($$$1 \le t \le 10^4$$$)&nbsp;— the number of test cases. The description of the test cases follows.</p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1 \le n \le 50$$$)&nbsp;— the number of outcomes.</p><p>The second line of each test case contains $$$n$$$ integers $$$k_1,k_2,\ldots,k_n$$$ ($$$2 \le k_i \le 20$$$)&nbsp;— the multiplier for the amount of coins if the $$$i$$$-th outcome turns out to be winning.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output $$$-1$$$ if there is no way to distribute the coins as required. Otherwise, output $$$n$$$ integers $$$x_1, x_2,\ldots, x_n$$$ ($$$1 \le x_i \le 10^{9}$$$)&nbsp;— your bets on the outcomes.</p><p>It can be shown that if a solution exists, there is always a solution that satisfies these constraints.</p><p>If there are multiple suitable solutions, output any of them.</p></div><div class="sample-tests"><div class="section-title">Example</div><div class="sample-test"><div class="input"><div class="title">Input</div><pre><div class="test-example-line test-example-line-even test-example-line-0">6</div><div class="test-example-line test-example-line-odd test-example-line-1">3</div><div class="test-example-line test-example-line-odd test-example-line-1">3 2 7</div><div class="test-example-line test-example-line-even test-example-line-2">2</div><div class="test-example-line test-example-line-even test-example-line-2">3 3</div><div class="test-example-line test-example-line-odd test-example-line-3">5</div><div class="test-example-line test-example-line-odd test-example-line-3">5 5 5 5 5</div><div class="test-example-line test-example-line-even test-example-line-4">6</div><div class="test-example-line test-example-line-even test-example-line-4">7 9 3 17 9 13</div><div class="test-example-line test-example-line-odd test-example-line-5">3</div><div class="test-example-line test-example-line-odd test-example-line-5">6 3 2</div><div class="test-example-line test-example-line-even test-example-line-6">5</div><div class="test-example-line test-example-line-even test-example-line-6">9 4 6 8 3</div></pre></div><div class="output"><div class="title">Output</div><pre>27 41 12

Then the total amount of coins bet on all outcomes is $$$27 + 41 + 12 = 80$$$ coins. If the first outcome turns out to be winning, you will receive back $$$3 \cdot 27 = 81$$$ coins, if the second outcome turns out to be winning, you will receive back $$$2 \cdot 41 = 82$$$ coins, if the third outcome turns out to be winning, you will receive back $$$7 \cdot 12 = 84$$$ coins.
